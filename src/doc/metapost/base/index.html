Index of texmf/doc/metapost/

This page provides access to documentation about the MetaPost system by John Hobby, including the MetaPost program and standard MetaPost macro packages.

Additional information about the MetaPost system can be found on John Hobby's WWW pages at http://plan9.bell-labs.com/who/hobby/MetaPost.html. Other links are available from TUG's MetaPost web page.

Index

The MetaPost System (PDF)
Introduction to MetaPost, and also an example document for using MetaPost with LaTeX.
A User's Manual for MetaPost (PDF)
AT&T Bell Labs, CSTR 162, by John Hobby, 1992.
The primary manual for the MetaPost system, which includes a description of most standard MetaPost macro packages.
Drawing Graphs with MetaPost (PDF)
AT&T Bell Labs, CSTR 164, by John Hobby, 1993.
The manual for the MetaPost graph.mp package and a few specialized features of the MetaPost program that were added after the publication of the user manual.
MetaPost with groff (PDF)
Example of using MetaPost with groff.
source
Subdirectory with all the MetaPost, TeX, and roff sources.
