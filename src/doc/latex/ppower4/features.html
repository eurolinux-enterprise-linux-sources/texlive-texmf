  • For the basics there is a demonstration document (German or English, both PDF format, created by pdflatex) and a report on PPower4 (again German or English, both PDF format)

  • More dynamic effects are shown in another demo (created by vtex on Linux). See the manual (below) for documentation and the download page for the source.
    If you want to use the extended Xfig to MetaPost export for animated figures, please download the LaTeX style file and see also our Xfig page.
    Depending on your Xfig version it may also necessary to upgrade your Xfig and transfig/fig2dev or to add some patches.
