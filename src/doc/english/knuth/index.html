TeX & METAFONT Manuals

These manuals are distributed only for people to see the examples of TeX input, not for use in the preparation of books like The TeXbook or The TeXbook. These manuals are copyrighted and should not be TeXed!

texbook.tex
This file is the TeX source of The TeXbook by Donald E. Knuth, last updated in August 1998.
mfbook.tex
This file is the TeX source of The METAFONTbook by Donald E. Knuth, last updated in August 1998.
mfman.mf
This file is the METAFONT source of most of the figures in the The METAFONTbook. It can be run through METAFONT and GFtoDVI to produce proof sheets.

TeX & METAFONT Examples

story.tex
This is the canonical example file for TeX which is described in Chapter 6 of The TeXbook.
io.mf
This is the canonical example file for METAFONT which is described in Chapter 5 of The METAFONTbook.
expr.mf
This is another example file for METAFONT which is described in Chapter 8 of The METAFONTbook.

TeX & METAFONT Errata

The Error Log of TeX (DVI)
This is an updated version of the appendix of the paper The Errors of TeX, last updated in March 1995. It includes a chronological listing of all changes and bug fixes in TeX (both TeX78 and TeX82) that have been applied since March 1978.
tex82.bug
This is a chronological listing of all changes and bug fixes in TeX (specifically TeX82) that have been applied since September 1982.
mf84.bug
This is a chronological listing of all changes and bug fixes in METAFONT (specifcally MF84) that have been applied since March 1984.
cm85.bug
This is a chronological listing of all changes and bug fixes in the Computer Modern fonts (CM85) that have been applied since April 1985.

Literate Programming

The WEB System of Structured Documentation (DVI)
This is an abridged version of the manual that describes the WEB system. Not included here are the WEB listings of the tangle and weave programs.
Literate Programming (DVI)
This is an online version of the article Literate Programming by Donald E. Knuth published in The Computer Journal 1984 that describes the philosophy of the WEB system.
primes.web
This is the WEB source of the example program discussed in the article Literate Programming. (A PASCAL program to print a table of prime numbers.)

Trip & Trap Test Manuals

A torture test for TeX (DVI)
This is an abridged version of the manual that describes the trip test, a torture test for TeX. (Most of the appendices have been left out since nobody really would want to read that anyway.)
A torture test for METAFONT (DVI)
This is an abridged version of the manual that describes the trap test, a torture test for METAFONT. (Most of the appendices have been left out since nobody really would want to read that anyway.)
